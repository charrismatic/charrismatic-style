# BASE THEME STRUCTURE AND LAYOUT CLASSES

### TYPICAL PAGE STRUCTURE

```
html > head > body  ...
 
  header
    |-- nav
  
  section [ .banner | .main ]  
    |-- .row
      |-- .sidebar [ if true ]
      |-- .content-block
        |-- .content
          |-- .row
          |-- .row
      |-- .content-block [other]
    |-- .row
   
   section [ .special ]
    |-- .row
    |-- .row
   
   section [ .contact | .others ]
    |-- .row
   
   footer
```
